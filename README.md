# dotfiles

My configuration files. They are meant to be used in a \*nix environment.

> [!WARNING]
> This repo is not meant for general use. It evolves on a daily basis and is only suitable for my own workflow.

## Usage

1. clone the repo to `~/dotfiles`
1. install brew and a few global tools, mainly wezterm, some fonts and mise
1. stow configs in the root directory to the home directory
1. install all the other tools with mise
1. autostart fish shell to bootstrap the custom shell environment
